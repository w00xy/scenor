## ai.scenor.ru — сервис генерации сценариев для видео

ai.scenor.ru помогает маркетологам, продюсерам и авторам быстро получать сценарии для роликов: от рекламных интеграций до обучающих видео. Проект стартует с базового генератора текстов и постепенно будет расширяться новыми инструментами для полного цикла подготовки контента.

### Текущие возможности
- Создание сценариев по краткому описанию или брифу
- Гибкие шаблоны с настройкой тона, длительности и CTA
- Экспорт текста и совместный доступ к сценариям

### План развития
- Дополнительные AI‑модули (расшифровки, резюмирование)
- Командные рабочие пространства, роли и общий медиаархив

### Архитектура
- Backend: `FastAPI` (Python) для REST/WS и фоновых задач
- Frontend: `React` + `SCSS` для SPA
- Хранилище: PostgreSQL + S3-совместимый стор для ассетов
- Очереди и фоновые задачи: Celery + Redis (планируется)
